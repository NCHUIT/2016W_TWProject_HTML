第一步：HTML5 基本版型

<!DOCTYPE html>
  <html lang="en">
  <head>
    <meta charset="UTF-8" />
    <title>Document</title>
  </head>
  <body>
  </body>
</html>

放些元素在 body 裡頭吧

  • 文字：h1 - h6 / p
  • 圖片：<img src="" />
  • 超連結：<a href="">...</a>

再加入些容器吧

  • 表格： table>tr*3>td*3
  • 清單： ol>li*3
  • 白板：<div>...</div>

CSS 簡單說明

  • 不懂？按下 F12 / CMD+Opt+i 打開開發者介面的第一個工具的右半邊那個就是
  • 對你的文字加個 color: red; 試試

The quickest http server

  1. 下載並安裝 NodeJS
  2. 打開 CMD ，輸入 npm install -g http-server
  3. 在 windows 上，按著 shift 按右鍵選 在此處開啟命令提示視窗 (或 cd 你的目錄) 然後輸入 http-server

Semantic UI 提供的一些比 HTML 原始好用元件

  • 比 HTML 內建變化多的 Header
  • 比 HTML 內建功能豐富的 Image
  • 比 HTML 內建超連結美的 Button

Semantic UI 提供的一些比 HTML 原始好用容器

  • 比 HTML 內建屌多的 Table
  • 比 HTML 內建屌猛的 List

Semantic UI 提供的一些原本 HTML 沒有的東西

  • Icon
  • Segment
  • 當然還有很多，要學著自己查資料啊
